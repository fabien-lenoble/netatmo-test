# Netatmo
## Web application development exercise

### Run the backend ([port 3001](http://localhost:3001/))

```sh
cd backend
npm i
node main.js
```
### Run the frontend ([port 5173](http://localhost:5173/))

```sh
cd frontend
npm i
npm run dev
```
